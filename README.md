# TVCurve

## 1. Overview

TVCurve is an R Shiny application for survival analysis with time-varying covariates.

The application was developed for the publication:

*A Novel R Shiny Tool TVCurve<sup>TM</sup> for Survival Analysis with Time-Varying Covariate in Oncology Clinical Studies: Overcoming Biases and Enhancing Collaboration*

## 2. Structure

| Component | Files | Description |
|-----------|-------|-------------|
| Shiny app | `server.R`, `ui.R` | Main application files |
| Shiny app | `deploy/` | Files required for deployment |
| Shiny app | `auth_example.txt` | Example authentication configuration |
| Shiny app | `www/TVCurve_1080p.mp4` | Demo video shown in the application |

*Note: Additional manuscript materials and protected datasets remain on the FH server.*

## 3. Run Shiny App

The application is available at: https://qiaoyang00.shinyapps.io/TVCurve/

To run locally:

```r
shiny::runApp()
```
